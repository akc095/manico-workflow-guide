## Manico 解决的是哪种"切换焦虑"

每个人切换 App 的习惯不一样。有人靠鼠标点 Dock，有人靠 Command+Tab 死磕，也有人从 Windows 换过来，怀念任务栏上 Win+数字键直接跳转的那套操作——Mac 原生是没有的。

Manico 提供三种模式，分别对应不同的人：

- **Dock 模式**：自动映射程序坞里的图标，装上就能用，不用配置，适合刚接触的新手
- **切换器模式**：映射当前打开的 App，相当于加强版 Command+Tab，适合同时开很多窗口的重度用户
- **自定义模式**：自己指定要用的 App 和对应快捷键，比如 Chrome 设成 Option+1、终端设成 Option+2，按自己的习惯走

除了这三种基本模式，它还带了几个细节功能：可以调整启动条出现的位置和指示器外观，可以把某些会用到 Option 键的软件（比如部分音乐制作软件、终端密码输入）加进"静默模式"列表避免冲突，还内置了用量统计，能看到自己每天调用了多少次软件。

说白了，它干的事情很单一：把"找到并打开目标 App"这件事，从一个需要眼睛盯屏幕的操作，变成一个手指肌肉记忆就能完成的动作。
